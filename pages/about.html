건축물대장 검색

주소만 입력하면 건축물대장 정보와 함께 건축허가일 기준 소방시설 설치기준을 확인할 수 있습니다.

법령 타임라인

2003년부터 현재까지 소방법령의 주요 변경 내용을 연도별로 정리하여 제공합니다.

용도별 체크리스트

30개 건물 용도별 소방시설 설치기준을 체크리스트 형태로 제공합니다.

소방시설 가이드

스프링클러, 소화기, 자동화재탐지설비 등 주요 소방시설의 설치기준을 상세하게 안내합니다.

서비스 소개

림정파이어는 소방 전문가, 건축주, 시설관리자를 위한 소방시설 설치기준 확인 서비스입니다.

건축물의 소방시설 설치기준은 건축허가일 당시의 법령을 적용합니다. 따라서 건축물대장에서 건축허가일을 확인하고, 해당 시점의 법령을 찾아보는 과정이 필요합니다.

림정파이어는 이 과정을 간소화하여, 주소 검색만으로 건축허가일 기준의 소방시설 설치기준을 빠르게 확인할 수 있도록 도와드립니다.

주의: 본 서비스에서 제공하는 정보는 참고용이며, 법적 효력이 있는 공식 문서가 아닙니다. 실제 소방시설 설치 시에는 반드시 관할 소방서 및 전문가와 상담하시기 바랍니다.

데이터 출처

림정파이어는 신뢰할 수 있는 공공 데이터를 기반으로 서비스를 제공합니다.

사용 데이터

  • 건축물대장 정보: 국토교통부 건축물대장정보 서비스 API
  • 소방법령 정보: 국가법령정보센터 (법제처)
  • 주소 검색: 카카오 주소 검색 API
  • 지도 서비스: 카카오 지도 API
